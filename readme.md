# Todos API with FastAPI and PostgreSQL

**A simple FastAPI application for managing todos. Users can create, read, update, and delete todos with a title, description, priority, and completion status.**

## EZ start

1. Clone the repository
2. Navigate to the project directory
3. run `docker compose up --build`
4. Open your browser and go to `http://localhost:4444` to see the application in action.

> tip: go to `http://localhost:4000/docs` to access the API documentation.
